cos_migrate_tool
##########################

建议使用 0.3.1 之后的版本
    
A simple tool for migrating data between object storage services.

INSTALL
-----------

use pip ::
    python setup.py install


HOW TO USE
---------------

as following ::

    cos_migrate_tool -h
    cos_migrate_tool -c /path/to/your/conf
